### Linux date string format 
### date : 2021-12-26
### tag  : \#Date

---

사유 : file 명 지정시 날짜 사용 

### 날짜 출력
```shell
   > date '+%Y-%m-%d'
   > 2021-12-26
```

### 날짜/시간 출력
```shell
   > date '+%Y-%m-%d_%H:%M'
   > 2021-12-26
```
